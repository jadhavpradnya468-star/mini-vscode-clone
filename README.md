#  Mini VS Code Clone

A powerful **browser-based code editor** built using **HTML, CSS, and JavaScript** that simulates core features of modern IDEs like VS Code.

---

## 🚀 Features

✨ Multi-file system (create & switch files)
🎨 Syntax highlighting (HTML, CSS, JS)
⚡ Autocomplete (Ctrl + Space)
🐞 Error highlighting (linting)
✨ Code formatting (Prettier)
🌗 Theme switch (Dark / Light)
📦 Live preview (run code instantly)
⬇️ Download project files
⌨️ Keyboard shortcuts (Ctrl + Enter, Ctrl + S)

---

## 🧠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* CodeMirror (Editor Engine)
* Prettier (Code Formatter)

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/mini-vscode-clone.git
```

2. Open the project:

```bash
cd mini-vscode-clone
```

3. Run:

* Open `index.html` in your browser

---

## ▶️ Usage

1. Create or select files from sidebar
2. Write code in editor
3. Press **▶ Run** or `Ctrl + Enter`
4. See output in preview panel
---

## 💡 How It Works

* Each file is stored in a JavaScript object (simulated file system)
* Code is combined and rendered using an `<iframe>`
* CodeMirror handles editing, highlighting, and autocomplete
* Prettier formats code dynamically
